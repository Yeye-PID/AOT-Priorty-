 Disclemoer shell :
- Shell script ini saya bikin untuk mengurangi beban pada Runtime dengan memanfaatkan AOT (Ahead-of-Time) pada PM < pacakge manager > 

Dengan cara :
- Mengganti Mode AOT dengan mode yang lebih cepat tanpa memakan beban CPU, GPU And RAM secara signifikan
- Membersikan cache Compilelasi yang rusak pada Runtime
- Mempercepat pembuatan bycode mesin tanpa verifikasi tambahan

Efek yang didapat ketika memakai script :
- Mempercepat loading aplikasi
- Mengurangi lag dan stutter pada game And multitasking
- Menstabilekan FPS 
- Mengurangi beban CPU And RAM selama aplikasi di execute
- Mempercepat pembuatan kode runtime 
And lain-lainya.....

Recomened :
- Gaming
- Multiasking
